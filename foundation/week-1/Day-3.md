# 1. WHAT IS MACHINE LEARNING?

Machine Learning (ML) is a branch of Artificial Intelligence (AI) that enables computers to learn from data and make predictions or decisions without being explicitly programmed.

### Simple Definition

> Machine Learning teaches computers to learn patterns from data and improve performance through experience.

---

## Why Machine Learning?

✔ Automates decision-making

✔ Finds hidden patterns in data

✔ Improves prediction accuracy

✔ Handles large-scale data efficiently

✔ Powers intelligent applications

---

## Real-World Examples

| Application | Example                 |
| ----------- | ----------------------- |
| Email       | Spam Detection          |
| E-Commerce  | Product Recommendations |
| Banking     | Fraud Detection         |
| Healthcare  | Disease Prediction      |
| Streaming   | Netflix Recommendations |

# 3. CORE TYPES OF MACHINE LEARNING

Machine Learning is mainly classified into three categories.

---

## 1. SUPERVISED LEARNING

### Definition

A learning method where the model is trained using labeled data.

### Formula

```text
Input Data + Correct Output
            ↓
       Model Learns
            ↓
      Future Prediction
```

### Example

| Input          | Output          |
| -------------- | --------------- |
| House Features | House Price     |
| Customer Data  | Churn/Not Churn |

### Common Algorithms

* Linear Regression
* Logistic Regression
* Decision Tree
* Random Forest
* Support Vector Machine (SVM)

### Applications

✔ Sales Prediction

✔ Spam Detection

✔ Credit Risk Analysis

✔ Medical Diagnosis

---

## 2. UNSUPERVISED LEARNING

### Definition

A learning method where the model works with unlabeled data and discovers hidden patterns.

### Formula

```text
Input Data
     ↓
Find Patterns
     ↓
Grouping / Clustering
```

### Example

Customer Segmentation:

```text
Customers
     ↓
Cluster Based on Behavior
     ↓
Groups Created
```

### Common Algorithms

* K-Means Clustering
* Hierarchical Clustering
* DBSCAN
* PCA

### Applications

✔ Customer Segmentation

✔ Market Basket Analysis

✔ Anomaly Detection

✔ Pattern Recognition

---

## 3. REINFORCEMENT LEARNING

### Definition

A learning method where an agent learns by interacting with an environment and receiving rewards or penalties.

### Formula

```text
Agent
   ↓
Action
   ↓
Environment
   ↓
Reward / Penalty
   ↓
Learning
```

### Example

* Self-driving Cars
* Chess Playing AI
* Robotics
* Game AI

### Applications

✔ Robotics

✔ Autonomous Vehicles

✔ Industrial Automation

✔ AI Gaming

---

# COMPARISON OF ML TYPES

| Feature | Supervised       | Unsupervised      | Reinforcement       |
| ------- | ---------------- | ----------------- | ------------------- |
| Data    | Labeled          | Unlabeled         | Environment         |
| Goal    | Prediction       | Pattern Discovery | Reward Maximization |
| Output  | Known Target     | Hidden Patterns   | Optimal Actions     |
| Example | Price Prediction | Customer Groups   | Self-Driving Cars   |
